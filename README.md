# Projet "essaie site ucad"

Petit projet d'exemples HTML/CSS utilisés pour les exercices de cours.

Fichiers principaux:
- `connexion.html` — page de connexion
- `inscription.html` — page d'inscription

Instructions rapides:
1. Ouvrir `connexion.html` dans un navigateur pour voir la page de connexion.
2. Modifier le CSS dans le fichier ou dans `style.css` si besoin.
